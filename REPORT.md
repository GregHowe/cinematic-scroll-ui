### Summary
This project recreates a cinematic, scroll-driven experience inspired by the Zero Limits website.  
A generative atmospheric particle field replaces the original 3D model to maintain visual depth and mood while keeping performance high.

### Key Technologies
- React + Vite
- Framer Motion
- react-tsparticles
