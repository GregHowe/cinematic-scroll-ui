### Summary
This project recreates a cinematic, scroll-driven experience inspired by the Zero Limits website.  
The 3D centerpiece was replaced with a generative particle atmosphere to maintain depth and immersion while keeping performance high.

### Key Technologies
- React + Vite
- Framer Motion (`useScroll`, `useTransform`)
- react-tsparticles (custom atmospheric particle field)
